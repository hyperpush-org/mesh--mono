Extend `build_project()` in `compiler/meshc/src/discovery.rs` to also discover and register `.mpl` files from packages installed in `.mesh/packages/*/`.

**Strategy:** After collecting the project's own files, check if `.mesh/packages/` exists. For each subdirectory (format: `{name}@{version}`), call `discover_mesh_files()` on the package root and add those files to the module graph with paths relative to the package install root.

Package files should be registered with module names derived from their file paths relative to the package root — same as the existing `path_to_module_name()` logic. This means `slug.mpl` at package root becomes module `Slug`, which is exactly what `from Slug import ...` requires.

**Important:** `main.mpl` in installed packages is NOT included (it's the package's own entry point, not meant for consumer imports). The `path_to_module_name()` function already returns `None` for `main.mpl`, but when reading the file into the graph we must skip `None` names.

Modify `build_project()`:

```rust
pub fn build_project(project_root: &Path) -> Result<ProjectData, String> {
    // Phase 1: Discover project files
    let files = discover_mesh_files(project_root)?;
    let mut graph = ModuleGraph::new();
    let mut module_sources = Vec::new();
    let mut module_parses = Vec::new();

    for relative_path in &files {
        // ... existing per-file logic unchanged ...
    }

    // Phase 1b: Discover installed package modules from .mesh/packages/*/
    let packages_dir = project_root.join(".mesh").join("packages");
    if packages_dir.exists() {
        for entry in std::fs::read_dir(&packages_dir)
            .map_err(|e| format!("Failed to read .mesh/packages: {}", e))?
        {
            let entry = entry.map_err(|e| format!("Failed to read packages entry: {}", e))?;
            let pkg_dir = entry.path();
            if !pkg_dir.is_dir() { continue; }

            let pkg_files = discover_mesh_files(&pkg_dir)?;
            for relative_path in &pkg_files {
                let name = match path_to_module_name(relative_path) {
                    Some(n) => n,
                    None => continue, // skip main.mpl
                };
                let full_path = pkg_dir.join(relative_path);
                let source = std::fs::read_to_string(&full_path)
                    .map_err(|e| format!("Failed to read '{}': {}", full_path.display(), e))?;
                let parse = mesh_parser::parse(&source);
                let _id = graph.add_module(name, relative_path.clone(), false);
                module_sources.push(source);
                module_parses.push(parse);
            }
        }
    }

    // Phase 2 and 3: Build dependency edges and topological sort (unchanged)
```

## Implementation Approach

### slugify/1 and slugify/2

Strategy using Regex:
1. `String.to_lower(str)` — lowercase everything
2. `Regex.replace(~r/[^a-z0-9]+/, lowered, sep)` — replace all non-alphanumeric sequences with sep
3. `String.trim(result)` followed by stripping leading/trailing sep — or use regex anchors

Note: `Regex.replace` replaces ALL matches. Using `[^a-z0-9]+` (one-or-more) as the pattern
collapses consecutive non-alphanumeric chars into one separator automatically.

After replacement, strip leading/trailing sep:
- If result starts with sep: strip it
- If result ends with sep: strip it
- The Regex.replace with `^sep|sep$` anchors can do both in one pass if Mesh Regex supports anchors
- Alternatively: two String.replace calls won't work for arbitrary sep; use a helper that checks starts_with/ends_with and slices

For stripping leading/trailing sep, implement a private helper:
```mesh
fn strip_sep(s :: String, sep :: String) -> String do
  let s1 = if String.starts_with(s, sep) do
    # slice off the leading sep
    String.split(s, sep) |> ... # or use Regex
  else s end
  # similar for trailing
  ...
end
```

Simpler approach: since sep defaults to "-" and custom sep is user-supplied, use Regex with
the sep as a literal to strip edges. But Regex.compile with dynamic patterns is available:
  `Regex.compile("^" <> sep <> "+|" <> sep <> "+$")`
This covers both leading and trailing in one replace.

If Regex.compile fails or is unavailable for dynamic patterns, fall back to:
  Split on sep -> filter out empty strings -> join back

The split-filter-join approach is robust:
1. lower = String.to_lower(str)
2. replaced = Regex.replace(~r/[^a-z0-9]+/, lower, sep)  -- static pattern, compile-time
3. parts = String.split(replaced, sep)
4. non_empty = List.filter(parts, fn(p) do String.length(p) > 0 end)
5. String.join(non_empty, sep)

This naturally handles leading/trailing/consecutive separators.

### truncate/2

Strategy:
1. If `String.length(slug) <= max`, return slug as-is
2. Take the first max characters: need a substring operation
   - Check if `String.split` + `String.join` can reconstruct a prefix
   - Or: check Mesh for `String.slice(s, 0, max)` or `String.take(s, max)`
   - If no direct substring: split by separator, accumulate words until length would exceed max
3. Find the last separator before the max boundary, cut there

Word-accumulation approach (avoids needing substring):
```mesh
fn truncate(slug :: String, max :: Int) -> String do
  if String.length(slug) <= max do
    slug
  else
    let parts = String.split(slug, "-")
    accumulate_words(parts, max, "", "-")
  end
end

fn accumulate_words(parts, max :: Int, acc :: String, sep :: String) -> String do
  case List.head(parts) do
    None -> acc
    Some(word) ->
      let candidate = if String.length(acc) == 0 do word else acc <> sep <> word end
      if String.length(candidate) > max do
        acc  # this word would exceed max, stop
      else
        let rest = List.tail(parts)  # or however List tail is accessed
        accumulate_words(rest, max, candidate, sep)
      end
  end
end
```

Note: Check whether List.tail exists in Mesh stdlib. If not, use List.drop(list, 1) or
implement via List.filter with an index pattern. Consult the compiler stdlib for available
List functions.

### is_valid/1

Use a compile-time Regex literal:
```mesh
fn is_valid(slug :: String) -> Bool do
  if String.length(slug) == 0 do
    false
  else
    Regex.is_match(~r/^[a-z0-9]+(-[a-z0-9]+)*$/, slug)
  end
end
```

Pattern `^[a-z0-9]+(-[a-z0-9]+)*$` matches:
- One or more alphanumeric chars
- Optionally followed by (hyphen + one-or-more alphanumeric) groups
- No leading/trailing hyphens, no consecutive hyphens

## Test File Structure

```mesh
# tests/slug.test.mpl
# Unit tests for mesh-slug package.
# Run with: meshc test mesh-slug

from Slug import slugify, truncate, is_valid

describe("Slug.slugify — default separator") do
  test("basic: lowercases and hyphenates") do
    assert_eq(slugify("Hello World!"), "hello-world")
  end
  test("consecutive spaces collapse to one separator") do
    assert_eq(slugify("foo  bar"), "foo-bar")
  end
  test("leading and trailing special chars stripped") do
    assert_eq(slugify("  hello  "), "hello")
  end
  test("empty string returns empty string") do
    assert_eq(slugify(""), "")
  end
  test("all-special input returns empty string") do
    assert_eq(slugify("!!!"), "")
  end
  test("already valid slug unchanged") do
    assert_eq(slugify("hello-world"), "hello-world")
  end
  test("long string with mixed content") do
    assert_eq(slugify("The Quick Brown Fox!"), "the-quick-brown-fox")
  end
end

describe("Slug.slugify — custom separator") do
  test("underscore separator") do
    assert_eq(slugify("Hello World!", "_"), "hello_world")
  end
  test("double-hyphen separator") do
    assert_eq(slugify("Hello World!", "--"), "hello--world")
  end
end

describe("Slug.truncate") do
  test("truncates at last separator boundary") do
    assert_eq(truncate("hello-world-foo", 11), "hello-world")
  end
  test("no truncation when within max") do
    assert_eq(truncate("hello-world-foo", 15), "hello-world-foo")
  end
  test("truncates to first word when max is small") do
    assert_eq(truncate("hello-world-foo", 5), "hello")
  end
  test("empty string returns empty string") do
    assert_eq(truncate("", 10), "")
  end
  test("single word shorter than max") do
    assert_eq(truncate("hello", 10), "hello")
  end
end

describe("Slug.is_valid") do
  test("valid slug returns true") do
    assert(is_valid("hello-world"))
  end
  test("single word returns true") do
    assert(is_valid("hello"))
  end
  test("digits allowed") do
    assert(is_valid("hello123"))
  end
  test("uppercase letters return false") do
    assert(is_valid("Hello-World") == false)
  end
  test("spaces return false") do
    assert(is_valid("hello world") == false)
  end
  test("leading hyphen returns false") do
    assert(is_valid("-hello") == false)
  end
  test("trailing hyphen returns false") do
    assert(is_valid("hello-") == false)
  end
  test("consecutive hyphens return false") do
    assert(is_valid("hello--world") == false)
  end
  test("empty string returns false") do
    assert(is_valid("") == false)
  end
  test("special chars return false") do
    assert(is_valid("hello!world") == false)
  end
end
```
